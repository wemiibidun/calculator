# Apple-Style Calculator (React + Vite)

A clean, Apple-inspired calculator built with React and Vite. The UI is intentionally minimal and polished, while the logic supports the core calculator behaviors expected in a real device.

## Preview

![Apple-style calculator preview](public/preview.svg)

## Features

- Apple-inspired layout, spacing, and visual styling
- Core calculator logic (AC/C, %, +/- , operations, equals)
- Keyboard support (digits, `+`, `-`, `*`, `/`, `.`, `Enter`, `Backspace`)
- Responsive sizing for mobile and desktop

## Tech Stack

- React 18
- Vite 5

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

- `index.html` — Vite HTML entry
- `src/main.jsx` — React entry
- `src/App.jsx` — Calculator UI + logic
- `src/App.css` — Component styling
- `src/index.css` — Global styling
